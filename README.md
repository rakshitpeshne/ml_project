# 🏥 Healthcare Cost Prediction

A Machine Learning-based web application that predicts healthcare/medical insurance costs based on user inputs like age, BMI, smoking habits, and other personal details.

---

## 📌 Project Overview

Healthcare costs vary depending on several factors such as age, lifestyle, and medical conditions. This project uses Machine Learning algorithms to predict medical insurance charges and provides insights into the factors affecting healthcare expenses.

The system combines:
- Data preprocessing & analysis
- Machine learning model building
- Web deployment using Flask

---

## 🎯 Objectives

- Predict healthcare/insurance costs using ML models  
- Analyze factors affecting medical expenses  
- Compare different regression algorithms  
- Deploy the model as a web application  

---

## 📊 Dataset

The dataset used in this project includes the following features:

| Feature     | Description |
|------------|------------|
| Age        | Age of the person |
| Sex        | Gender |
| BMI        | Body Mass Index |
| Children   | Number of dependents |
| Smoker     | Smoking status |
| Region     | Residential region |
| Charges    | Medical cost (target variable) |

---

## 🔍 Project Workflow

1. Data Collection

## 🤖 Machine Learning Models

- Linear Regression  
- Ridge Regression  
- Lasso Regression  
- Decision Tree Regressor  
- Random Forest Regressor  

---

## 📈 Results

- Random Forest gives the best performance  
- Important factors affecting cost:
  - Smoking  
  - BMI  
  - Age
